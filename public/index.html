Generador de Órdenes

Generador de Datos para Pruebas y Simulaciones

Guía de Uso

1. Generar Órdenes

Crea un archivo Excel con un listado de pedidos para importar en PlannerPro.

  • Geo-referenciación: Puedes filtrar clientes por País y Ciudad. El sistema asignará coordenadas lat/long reales de nuestra base de datos.
  • Round Robin: Si eliges una ciudad, los pedidos se distribuirán aleatoriamente entre los puntos de esa ciudad.
  • Tags Dinámicos: Agrega columnas extra (ej: Tipo_Servicio, Prioridad) con valores aleatorios.

2. Generar Vehículos

Crea la flota necesaria para tus simulaciones.

  • Configuración Mixta: Define cuántos camiones, autos o motos necesitas.
  • Datos por Defecto: El sistema rellena automáticamente costos, velocidades y restricciones (con valores realistas o vacíos según el estándar) para que solo tengas que preocuparte por la cantidad y el tipo.
  • Origen Obligatorio: Cada grupo de vehículos debe tener un Centro de Distribución (Origen) asignado.
